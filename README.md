# Schedula-Prince 🚀

A modern backend scheduling application built with [NestJS](https://nestjs.com/).

## 📖 Description

Schedula-Prince is a robust backend service designed for scheduling and time management. This project is built using the NestJS framework and TypeScript to ensure a scalable and maintainable architecture.

## 🛠️ Project Setup

```bash
# Install dependencies
$ npm install
```

## 🚀 Running the App

```bash
# Development mode
$ npm run start

# Watch mode (Recommended for active development)
$ npm run start:dev

# Production mode
$ npm run start:prod
```

By default, the application runs on **port 3000**. Once the server starts, you can access it at: `http://localhost:3000`

## 🧪 Testing

```bash
# Unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# Test coverage
$ npm run test:cov
```

## 📦 Tech Stack
- **Framework**: [NestJS](https://nestjs.com/) v11
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Testing**: Jest
